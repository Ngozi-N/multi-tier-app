About This Project

This project demonstrates a robust DevOps pipeline for a multi-tier web application, showcasing the integration of industry-leading tools:

  • Terraform: For declarative infrastructure provisioning on AWS.
  • Jenkins: Orchestrating continuous integration and continuous delivery.
  • Prometheus: A powerful monitoring system for collecting time-series data.
  • Grafana: For visualizing metrics and creating actionable dashboards.

Our goal is to provide a comprehensive example of how these tools work together to build, deploy, and maintain a highly available and observable cloud application.
